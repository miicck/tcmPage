Variational quantum monte carlo code

I am the author of a simple variational quantum monte carlo (VQMC) code which can be found here. A summary of VQMC and some results obtained using this code can be found here. The code can be used to construct a multi-electron wavefunction from atomic basis sets, and calculate the resulting variational energy in the presence of nucleii of your choosing. It will automatically enforce the correct exchange symmetry via a slater determinant and attempt to recover some of the correlation energy using a Jastrow factor. It works quite well for atomic and small molecular systems. Monte carlo is very easy to parallelise so there is a parallel branch (using MPI), as well as a serial version.
